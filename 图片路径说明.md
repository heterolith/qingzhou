# 晴宙博客 - 图片路径说明

## 图片存储位置

本项目使用 Vite 构建，图片按类别分文件夹存放在 `public` 目录下。

```
public/
├── favicon.ico              # 网站图标 (已存在)
├── avatar/                  # 头像类
│   └── avatar.jpg           # 头像图片 (需添加)
├── carousel/                # 轮播图类
│   ├── slide1.jpg           # 轮播图1 (需添加)
│   ├── slide2.jpg           # 轮播图2 (需添加)
│   └── slide3.jpg           # 轮播图3 (需添加)
├── works/                   # 作品/文章封面类
│   └── ...                  # (从数据库获取)
└── resources/               # 资源图标类
    └── qing.png             # 默认资源图标 (需添加)
```

## 如何修改图片路径

### 方法一：直接替换文件（最简单）

将你的图片文件放入 `public` 目录，使用相同的文件名覆盖：
- `avatar.jpg` - 替换头像
- `slide1.jpg`, `slide2.jpg`, `slide3.jpg` - 替换轮播图

### 方法二：修改代码引用

如果你想使用不同的文件名或目录结构，需要修改对应的 Vue 文件：

**示例 - 修改头像路径**：
```javascript
// src/components/layout/Sidebar.vue 第71行
avatar: 'avatar/my-avatar.jpg'  // 改为你的路径

// src/views/AboutView.vue 第103行
avatar: 'avatar/my-avatar.jpg'

// src/views/HomeView.vue 第510行 (CSS)
background-image: url('avatar/my-avatar.jpg');
```

**示例 - 修改轮播图路径**：
```javascript
// src/views/HomeView.vue
{
  // ...
  image: 'carousel/slide1.jpg'  // 改为你的路径
}
```

## 图片尺寸建议

| 图片类型 | 建议尺寸 | 格式 |
|---------|----------|------|
| 头像 | 400x400px (正方形) | JPG/PNG |
| 轮播图 | 800x600px (4:3) 或 600x400px | JPG/WebP |
| 作品封面 | 1200x630px (OG标准) | JPG/WebP |
| 资源图标 | 64x64px 或 128x128px | PNG/SVG |
| 网站图标 | 32x32px 或 64x64px | ICO/PNG |
