图片存储位置

分文件夹管理

本项目使用 Vite 构建，图片按类别分文件夹存放在 public/ 目录下：

public/ ├── favicon.ico # 网站图标 (已存在) │ ├── avatar/ # 头像类 │ └── avatar.jpg # 头像图片 (需添加) │ ├── carousel/ # 轮播图类 │ ├── slide1.jpg # 轮播图1 (需添加) │ ├── slide2.jpg # 轮播图2 (需添加) │ └── slide3.jpg # 轮播图3 (需添加) │ ├── works/ # 作品/文章封面类 │ └── ... # (从数据库获取) │ └── resources/ # 资源图标类 └── qing.png # 默认资源图标 (需添加)
👤

头像类图片

文件位置 变量名 默认值 用途
